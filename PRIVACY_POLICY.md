# 隐私政策 (Privacy Policy)

**最后更新日期：2026年5月29日**

欢迎使用 **AiWord**。我们深知隐私对您的重要性，因此我们制定了本隐私政策，以最透明、最清晰的方式向您说明我们如何处理您的数据。

**核心承诺：AiWord 是一款“本地优先 (Local-First)”的应用程序。我们不会收集、存储、分享或出售您的任何个人隐私数据。**

---

### 1. 我们收集什么信息？
**我们不收集任何您的个人信息。**
- 我们**不**要求您注册账号或提供手机号、邮箱等个人标识符。
- 我们**不**收集您的设备型号、IP 地址、广告标识符 (IDFA/OAID) 等设备信息。
- 我们**不**植入任何第三方追踪器、数据分析或广告 SDK（无诸如 Google Analytics、友盟等收集行为）。

### 2. 您的查词数据存放在哪里？
**您在 App 内产生的所有数据均完全保存在您的本地设备上。**
- 您的查词历史记录
- 您的“收藏夹”单词和划卡进度
- 您添加的情景模式 (Contexts) 配置
- 您的 API Key 及底层引擎设置
以上所有信息均通过本地沙盒机制安全地存储在您的手机存储器中。除非您主动卸载应用或清除应用数据，否则这些数据不会离开您的设备。

### 3. API 请求与第三方服务
如果您在设置中配置了第三方大语言模型 (LLM) 的 API（例如 DeepSeek、OpenAI 等），您的查词请求（仅包含您输入的查询词和情景 Prompt）将通过加密的 HTTPS 连接直接发送给对应的 API 服务商。
- 该过程**不会**经过我们的服务器中转。
- 请您留意所使用 API 服务商（如 OpenAI）本身的隐私政策，它们通常有明确的政策表明不会使用您的 API 查词请求来训练模型。

### 4. 设备权限调用说明
为实现应用的正常运行，我们可能需要使用以下本地权限，但均**不涉及您的隐私泄露**：
- **存储权限 (Storage/Files)**：仅当您主动使用“从本地导入 JSON/Bin 词库文件”功能时，系统会弹出文件选择器，应用仅会读取您选择的那个特定词库文件，绝不会扫描您的相册或文件目录。
- **网络权限 (Internet)**：仅用于向您配置的大语言模型发起查词请求，或当您点击“从 URL 下载词库更新”时获取词库文件。

### 5. 政策变更
因为我们根本不收集您的数据，所以本政策的核心承诺永远不会改变。如果未来由于添加了全新的网络功能而需要更新隐私政策，我们会在此页面更新并在应用内通知您。

### 6. 联系我们
如果您对本隐私政策或应用的数据处理机制有任何疑问，由于我们没有收集您的任何联系方式，建议您直接通过 GitHub 仓库的 Issues 页面与开发者取得联系。
